# 网页采集与表单自动化

用于网页登录、资料采集、后台检查、表单录入、截图留档。

## 目标

- 先读懂页面，再动手
- 把重复点击变成 SOP
- 提交前先检查，提交后留痕

## 标准流程

1. 打开页面
2. 读取页面结构和关键区域
3. 明确当前任务属于哪种：
- 信息采集
- 数据核对
- 表单填写
- 后台导出
- 结果截图

4. 操作前先检查：
- 页面是否完全加载
- 是否已登录
- 是否有弹窗
- 是否为不可逆动作

5. 不可逆动作必须确认：
- 提交
- 删除
- 发布
- 批量导出覆盖

## 留痕要求

- 关键节点截图
- 记录当前 URL
- 记录输入内容摘要
- 记录最终结果

## 失败处理

- 如果浏览器自动化异常，立即退回手动指导
- 如果页面反爬或风控，先提示切人工或更换模式
- 不要盲点，不要连续重复提交
