# 小红书运营自动化

用于小红书内容运营、爆款拆解、评论整理、选题生成、发布前准备。

## 目标

- 快速采集内容和评论
- 拆出可复用的爆点
- 生成标题、封面文案、正文框架
- 发布前做一遍风险检查

## 标准流程

1. 先确认是否已登录
2. 读取目标笔记或目标账号
3. 提取以下内容：
- 标题
- 封面文案
- 核心观点
- 评论高频问题
- 可模仿结构

4. 输出 3 类结果：
- 数据摘要
- 爆点拆解
- 新稿草案

## 自动化边界

- 可自动做采集、整理、起草
- 可自动做标题多版本生成
- 默认不自动发布
- 需要发布时，先做“发布前检查清单”

## 发布前检查清单

- 标题是否足够明确
- 开头 3 行是否抓人
- 是否有明确 CTA
- 是否涉及敏感词
- 图片/封面是否齐全
